Detailed Description

This header file contains definitions of structures and functions relating to loading obj files.

The term "obj file" should not be confused with the term "obj format file", as the former refers to exclusively Wavefront obj files, and thus to only geometry data. The latter can refer to both geometry data and any other generic data, such as shader configurations.

typedef struct lise_obj_mesh lise_obj_mesh

A structure that stores mesh data.

A "mesh" is a collection of unique vertices and indices which reference those vertices. Each mesh can only have one material.
